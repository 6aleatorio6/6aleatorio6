<h1>Olá! Bem-vindo ao meu perfil.👋</h1>

Sou apaixonado por programação, sempre buscando aprender novas habilidades e aplicá-las em projetos incríveis!

- 🤔 Participei de um projeto comunitário em parceria com a Prefeitura de Caraguatatuba-SP:
    - Desenvolvimento de um aplicativo em React Native para o Parque Juqueriquerê, visando melhorar a experiência dos visitantes e a gestão do parque.
    - Implementação do backend em Express, garantindo uma comunicação eficiente e segura entre o aplicativo e o servidor.
    - Criação de um sistema Admin/Totem em React para facilitar a gestão do parque e a interação com os usuários.
    - [Leia mais sobre o projeto aqui](https://www.caraguatatuba.sp.gov.br/pmc/2024/06/prefeitura-de-caraguatatuba-recebe-alunos-do-ifsp-para-apresentacao-de-aplicativo-para-o-parque-juqueriquere/).
